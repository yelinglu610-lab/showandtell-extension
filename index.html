你需要的，全在这里

🔦

激光笔

切换到激光模式，在页面上画出高亮轨迹，700ms 自动渐隐。支持 8 种颜色和自定义粗细。

📷

摄像头叠加

开启摄像头，人脸悬浮在页面上。边缘拖动、滚轮缩放、形状切换（圆角/圆形/方形）。

🔴

一键录制

录制屏幕 + 麦克风，包含激光轨迹和摄像头画面。停止后直接下载 webm 文件。

🖱

鼠标/激光切换

随时在鼠标模式（正常操作页面）和激光笔模式之间切换，快捷键 L / M。

🎙

麦克风控制

独立麦克风开关，图标灰色 = 关闭，黄色高亮 = 开启，状态一目了然。

∧

随手收起

点收起按钮，工具栏滑到右边缘变成小标签，不影响页面。点一下弹回来。

使用方法

三步开始演示

1

安装插件

从 GitHub 下载，打开 chrome://extensions/ 开启开发者模式，加载已解压的扩展程序。

2

点击图标激活

打开任意网页，点击 Chrome 工具栏的 ShowAndTell 图标，页面底部出现悬浮工具栏。

3

开始演示

开摄像头、打开激光笔、点录制。结束后点停止，录制文件自动下载到本地。
